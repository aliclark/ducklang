## Install

* Install a recent stable version of Node.js and NPM
* Review the [install script](https://raw.githubusercontent.com/ducklang-community/ducklang/main/install.sh)
* Run the install script:

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ducklang-community/ducklang/main/install.sh)"
```

## Principles

* Duck typing
* Testing not typing
* Encapsulation
* Composition not inheritance
* Inversion of control
* Readability not brevity
* Promises not channels
* Additional inputs are named not positional
* Extensibility of operators
* Separation of concerns
* Uniformity
* Error propagation
* Streams not lists
